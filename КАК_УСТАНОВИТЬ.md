# Установка на Samsung Galaxy S25 Ultra

## Шаг 1 — Установить Node.js
1. Перейти на https://nodejs.org и скачать LTS-версию
2. Установить с дефолтными настройками

## Шаг 2 — Установить зависимости
Открыть PowerShell в папке проекта:
```
npm install
```

## Шаг 3 — Запуск в режиме разработки (для теста)
```
npm start
```
Установить на телефон **Expo Go** из Google Play Store.
Отсканировать QR-код — приложение сразу откроется на телефоне.

## Шаг 4 — Сборка APK (для постоянного использования)

### 4.1 Зарегистрироваться на expo.dev (бесплатно)
### 4.2 Установить EAS CLI:
```
npm install -g eas-cli
eas login
eas build:configure
```
### 4.3 Собрать APK:
```
eas build --platform android --profile preview
```
Через 5-15 минут получите ссылку на скачивание APK.

### 4.4 Установить APK на телефон:
1. На телефоне: Настройки → Безопасность → Разрешить установку из неизвестных источников
2. Скачать APK и открыть — Android сам установит

## Что умеет приложение
- Учет доходов и расходов по категориям
- 15 встроенных категорий (можно добавлять свои)
- Общий баланс + статистика по месяцам
- График по месяцам за год
- Поиск и фильтрация операций
- Экспорт всех данных в CSV (открывается в Excel)
- Работает полностью офлайн, данные хранятся на телефоне
